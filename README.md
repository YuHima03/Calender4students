## 具体的な機能
###### 変更になる可能性があります
* 普通のカレンダーの機能は勿論
* カレンダーと様々なものを連携可能
  * 学校生活の管理
    * 時間割の管理
      * これがこのカレンダーの一番の機能
    * 時間割に合わせた課題の管理
      * 進捗状況も設定＆確認できるように
    * 考査の管理
      * テスト期間や範囲の設定＆確認できるように
    * 長期休暇の予定や課題の管理
    * 学校行事の管理
  * お金の管理
    * 収入と支出の管理
  * 番組表の管理
    * 見たいテレビ番組、アニメ等を見逃さないように
